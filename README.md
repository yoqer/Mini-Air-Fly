# Mini-Air-Fly

El Mini-Air-Fly es un Dron con forma cilíndrica de pocas dimensiones que compite con los drones con la estabilidad y bajo consumo energético del hidrógeno y el agua. Con el peso y tamaño de un refresco puede pasar desapercibido en estanterías y transporte. Pero cuenta con avanzado radar, altitud de vuelo limitada a resistencia y comunicacion de KMs.
